### ssh

```
ssh-keygen
```

主->从

```
cd /root/.ssh/
cat id_rsa.pub >> authorized_keys
chmod 600 authorized_keys 
```

set

```
scp master:/root/.ssh/id_rsa.pub /root
ls /root/
cat /root/id_rsa.pub  >> authorized_keys 
```

从->主

```
scp slave1:/root/.ssh/id_rsa.pub /root
cat /root/id_rsa.pub  >> authorized_keys
```
